Seasoning Shack Restaurant

  • JavaScript
  • HTML5
  • CSS
  • Adobe Photoshop
  • Adobe Illustrator
View Project

The Seasoning Shack is a fictitious restaurant that needed a website to showcase their business. Out of our group of developers I was responsible for the Event booking, Location and Home pages. The main focus was to explore our use of the JavaScript language. My pages include an automated promotion slider, form validation, store hours that update depending on current day and time, an accordion style image gallery, and a responsive navigation header. I also helped with the design and photo editing.

Space Colonies

  • JavaScript
  • HTML5
  • CSS
  • Adobe Illustrator
View Project

I joined a team of fellow web developers at the Global Game Jam 2018. We made an online 2‐Player game that requires each player to claim as many planets as possible before their opponent. I worked on the code behind populating values and styling the Heads-up Display, the planet detail mouseover feature, the initial splash page, as well as preparing suitable images for our team to use. If you'd like to try playing our game, ensure you and your opponent open the game in each other's browser. Or play against yourself by opening the game in 2 browser windows.
